# rainn_app

A new Flutter project.

## 🚀 Getting Started

This project is a starting point for a Flutter application.

### 📦 Cài đặt môi trường

Đảm bảo bạn đã cài:

* Flutter SDK
* Android Studio / VS Code
* Emulator hoặc thiết bị thật

Kiểm tra:

```bash
flutter doctor
```

---

## 📚 Cài đặt thư viện cần thiết

Chạy các lệnh sau trong terminal tại thư mục project:

### 🔥 Firebase (Core + Authentication + Firestore)

```bash
flutter pub add firebase_core
flutter pub add firebase_auth
flutter pub add cloud_firestore
```

---

### 🧭 Điều hướng (Routing)

```bash
flutter pub add go_router
```

---

### 📁 Chọn file

```bash
flutter pub add file_picker
```

---

### 🔐 Đăng nhập Google

```bash
flutter pub add google_sign_in
```

---

### 📊 Vẽ biểu đồ

```bash
flutter pub add fl_chart
```

---

### 🗺️ Google Maps

```bash
flutter pub add google_maps_flutter
```

---

### ⚙️ Firebase CLI (cài global)

```bash
npm install -g firebase-tools
```

---

### 📦 Cài đặt thư viện Backend (Node.js)

```bash
npm install express cors dotenv axios
npm install firebase-admin
npm install crypto-js
```

---

### ➕ Các thư viện Flutter bổ sung

```bash
flutter pub add image_picker
flutter pub add table_calendar
flutter pub add intl
flutter pub add firebase_storage
flutter pub add webview_flutter
```
## ▶️ Chạy project

```bash
flutter pub get
flutter run
```
